Perguntas frequentes

O que é o Apollo Traces?

O Apollo Traces é a forma mais inteligente e prática para rastreio Correios e outras empresas. O Apollo Traces funciona de forma automatizada, e monitora constantemente os pacotes que você cadastrar na sua conta. Todas as vezes que é detectado uma mudança no rastreio do pacote, o Apollo Traces te avisa por e-mail ou pelo melhor aplicativo de rastreamento, disponível para Android, iOS e Windows Phone.

É facil rastrear objetos pelo Muambator?

O Apollo Traces é a melhor opção para rastrear pacotes. E você precisa fazer só uma coisa: cadastrar o código de rastreio. Deixa todo o resto conosco. Simples, fácil, inteligente e grátis!

Como é um código de rastreio dos Correios?

Um código de rastreio emitido pelos Correios, tem o seguinte formato de apresentação: LLNNNNNNNNNBR. São duas letras, seguidas por nove números mais duas letras com final BR. As primeiras duas letras indicam o tipo da encomenda (se é Sedex, PAC, Carta Registrada). Os oito primeiros números são o código de identificação da encomenda. O nono número, é um dígito verificador, muito semelhante aos dois últimos dígitos de um CPF e/ou CNPJ.
